zim.main

package documentation
(source)

This module defines the main() function for executing the zim application. It also defines a number of command classes that implement specific commandline commands and an singleton application object that takes core of the process life cycle.
Module command This module is responsible for commandline parsing. It provides a base class for the commandline commands defined in zim.main
Module ipc This module is responsible for the inter-process communication (ipc) between zim instances.
