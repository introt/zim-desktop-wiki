zim.gui.widgets.NamespaceEntry

class NamespaceEntry(PageEntry): (source)

View In Hierarchy

Widget to select a zim page path as a namespace

Use this instead of PageEntry when you want to allow selecting a namespace. E.g. this will be allowed to select ":" or empty string for the root namespace, which is not allowed in PageEntry.
