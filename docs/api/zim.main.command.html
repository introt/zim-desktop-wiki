This module is responsible for commandline parsing. It provides a base class for the commandline commands defined in zim.main
Class Command Base class for commandline commands, used by zim to abstract part of the main() functionality and allow better testability of commandline arguments.
Class GtkCommand Base class for commandline commands that result in a Gtk user interface being presented to the user or want to interact with the Gtk user interface.
