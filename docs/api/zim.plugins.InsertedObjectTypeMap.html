class InsertedObjectTypeMap(SignalEmitter): (source)

View In Hierarchy

Mapping of InsertedObjectTypeExtension objects. This is a proxy for loading object types defined in plugins. For convenience you can use PluginManager.insertedobjects to access an instance of this mapping.
