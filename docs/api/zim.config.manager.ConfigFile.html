class ConfigFile(ConnectorMixin, SignalEmitter): (source)

View In Hierarchy

Container object for a config file

Maps to a "base" file in the home folder, used to write new values, and an optional default file, which is used for reading only.

Notethis class implement similar API to the File class but is explicitly not a sub-class of File because config files should typically not be moved, renamed, etc. It just implements the reading and writing methods.
