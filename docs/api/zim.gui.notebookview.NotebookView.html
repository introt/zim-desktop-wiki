zim.gui.notebookview.NotebookView

class documentation

class NotebookView(PageView): (source)

View In Hierarchy

Sub-class of the PageView class that is used when the view is navigatable. Plugins can choose to extend this class if they don't want to load extensions for a fixed page view.
