def user_dirs(): (source)

Get the XDG user dirs.

Returnsa dict with directories for the XDG user dirs. These are typically defined in "~/.config/user-dirs.dirs". Common user dirs are: "XDG_DESKTOP_DIR", "XDG_DOWNLOAD_DIR", etc. If no definition is found an empty dict will be returned.
