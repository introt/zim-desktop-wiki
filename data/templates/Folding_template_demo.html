Folding template

Created Saturday 04 December 2021

What is this?

This is an example page for the folding_with_index template, demonstrating the possibilities of the interactive yet JavaScript free <summary> HTML element.

It's not meant to be pretty.

Index

The index has been moved from the side to up top, giving the page contents more room. It could open to the side, moving the page contents, but the more pressing issue is the fact that its tree view doesn't fold. I'm not sure if this can be achieved with templates alone, but regardless of the possible template hacks I'd like to see it implemented in the HTML exporter itself.
